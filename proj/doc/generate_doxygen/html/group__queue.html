Detailed Description

Handles a Queue of uint8_t data types.

Queue implementation using linked list in C

Note
Inspired by https://codeforwin.org/2018/08/queue-implementation-using-linked-list-in-c.html
This original implementation was REALLY altered
It was full of bugs, you could only have one queue per program, the datatype was wrong, it has major bugs (memory leaks, wrong pointers that impacted the rest of the program), it was only two nodes (front and rear) that you had to play around with both of them as arguments
TL:DR
It was buggy and impossible to work with, so we added a LOT of code, fixed all its bugs and implemented a proper Queue struct
