Draws a Bitmap on the screen.

Upon rendering, the image will be 'stretched' (reproduced) to fulfill the indicated size

Warning
For optimal results, the bitmap should be square and have a lenghth that is a multiple of 3, the length / 3 will give the ideal dynamic_slice_size
Parameters
bmpPointer to the Bitmap to be rendered
dynamic_slice_sizeThe slice size
xThe x coordinate to draw on
yThe y coordinate to draw on
widthThe width to draw
Calls to render a width lesser than twice the object's dynamic_slice_size will be ignored
heightThe height to draw
Calls to render a height lesser than twice the object's dynamic_slice_size will be ignored

multiplyThe RGB 565 color to have a bitwise & operation with each pixel
