Typedef Documentation

◆ PowerUp_t

typedef struct PowerUp PowerUp_t

Represents a power up on a Level, such as unlocking the player's ability to change lasers or activate anti-gravity, and also the Level's exit.

Every power up has a void function pointer that is called whenever it is collected

Note
Could be expanded to create coin like collectibles on a Level
