◆ Lasers_t

typedef struct Lasers Lasers_t

Structure that represents all the Lasers in a Level.

Every Lasers object has a current link ID, when an individual Laser's link ID is the same as the Lasers' one, it will not be rendered and it won't be considered harmful to the player
