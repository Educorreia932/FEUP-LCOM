◆ Sprite_t

typedef struct Sprite Sprite_t

Renders a bitmap to the screen, while allowing it to be animated.

This is done by allowing up to 255 bitmaps to be loaded at once, and having a mutable state that decides which of these bitmaps to render. Clever use of this state alonside a state machine allows you to create custom animated Sprites.
All calls to the renderer allow you to reverse the image along any of its two axis

◆ SpriteDynamic_t

typedef struct SpriteDynamic SpriteDynamic_t

Renders a bitmap onto the screen, multiplying parts of the bitmap to fulfill the required size.

Warning
For optimal results, the bitmap should be square and have a lenghth that is a multiple of 3, the length / 3 will give the ideal dynamic_slice_size
