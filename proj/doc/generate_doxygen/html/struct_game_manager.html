Detailed Description

The class responsible for handling everything. Instanced as a Singleton to provide acess from anywhere on the files.

This is responsible for switching between menus and gamemodes and some of the most important variables to guarantee that two PCs are still connected via serial port
Only one of the level, s_board or main_menu is actually instantiated at a time, meaning that when they are not in use, they are NULL pointers.
