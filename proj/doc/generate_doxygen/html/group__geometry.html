Typedef Documentation

◆ Rect_t

typedef struct Rect Rect_t

Represents a rectangle in a plane.

The rectangle is represented by its top leftmost point (smallest x and y coordinates) and its size (width and height)

Note
This is used for various purpuoses, such as:
Colliders, sprite sizes, UI element's "hitbox", etc...

◆ Vec2d_t

typedef struct Vec2d Vec2d_t

Structure to represent two dimensional vectors.

Note
A point is also represented as a vector, because a point can be seen as the vector from the origin of the referencial to the point
